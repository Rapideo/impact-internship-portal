# Sub-Project 1: Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Empty production app deploys to Netlify with the full database schema, Row-Level Security policies, dev seed data, and a working login flow that routes admins and employers to placeholder dashboards. No real admin/employer features yet — those land in sub-projects 2-5.

**Architecture:** A new React Router v7 (framework mode) TypeScript app lives under `app/` in the existing repo (sibling of `Prototypes/`). Database is Supabase Postgres with all 15 tables defined via Drizzle and RLS policies enforcing the three-tier permission model (admin / employer / anonymous intern). Auth uses Supabase Auth with a Custom Access Token Hook that adds `role` and `employer_id` JWT claims, both consumed by RLS policies and by app-side route guards. Email transport via Resend. CI runs lint + typecheck + Vitest + Playwright on every PR; Netlify deploys on push to master.

**Tech Stack:** TypeScript 5.7, React Router v7 (framework mode), Vite 6, Drizzle ORM 0.36, postgres-js 3.4, Supabase JS 2.46 (with `@supabase/ssr` 0.5), Resend 4, Vitest 2, Playwright 1.49, ESLint 9 (flat config), Prettier 3, GitHub Actions, Netlify.

**Spec:** `docs/superpowers/specs/2026-05-10-production-rebuild-design.md`

**Working directory for all paths below:** `IMPACT Internship Assessment Portal/` (the repo root, after the Phase A folder rename).

  ```tsx
        </body>
      </html>
    );
  }

  export default function App() {
    return <Outlet />;
  }

  export function ErrorBoundary({ error }: Route.ErrorBoundaryProps) {
    let message = 'Something went wrong';
    let details = 'An unexpected error occurred.';
    let stack: string | undefined;

    if (isRouteErrorResponse(error)) {
      message = error.status === 404 ? 'Not found' : `Error ${error.status}`;
      details =
        error.status === 404
          ? 'The page you requested does not exist.'
          : error.statusText || details;
    } else if (import.meta.env.DEV && error instanceof Error) {
      details = error.message;
      stack = error.stack;
    }

    return (
      <main className="error-page">
        <h1>{message}</h1>
        <p>{details}</p>
        {stack && <pre>{stack}</pre>}
      </main>
    );
  }
  ```

  ```ts
  /* ============================================================ */
  ```

We don't use drizzle-kit to manage RLS because policy syntax is more readable in raw SQL files. A small Node script applies them in order.

- [ ] **Step 1: Create `db/apply-policies.ts`**

  ```ts
  import 'dotenv/config';
  import postgres from 'postgres';
  import { readdir, readFile } from 'node:fs/promises';
  import { join } from 'node:path';

  const POLICIES_DIR = join(process.cwd(), 'db', 'policies');
  const databaseUrl = process.env.DATABASE_URL;

  if (!databaseUrl) {
    console.error('DATABASE_URL is required');
    process.exit(1);
  }

  async function main() {
    const files = (await readdir(POLICIES_DIR))
      .filter((f) => f.endsWith('.sql'))
      .sort();

    if (files.length === 0) {
      console.log('No policy files found in db/policies/');
      return;
    }

    const sql = postgres(databaseUrl, { max: 1 });

    try {
      for (const file of files) {
        const path = join(POLICIES_DIR, file);
        const content = await readFile(path, 'utf-8');
        console.log(`Applying ${file}...`);
        await sql.unsafe(content);
      }
      console.log(`Applied ${files.length} policy file(s).`);
    } finally {
      await sql.end();
    }
  }

  main().catch((err) => {
    console.error(err);
    process.exit(1);
  });
  ```

### Task 30: Write the JWT Custom Access Token Hook

**Files:**
- Create: `db/policies/0004_jwt_hook.sql`

This Postgres function is invoked by Supabase Auth on every JWT issuance to inject `role` and `employer_id` claims.

- [ ] **Step 1: Create the file**

  ```sql
  -- Custom Access Token Hook: copy profile.role + profile.employer_id into JWT claims.
  -- Registered in Supabase Dashboard → Auth → Hooks → "Custom Access Token Hook".

  CREATE OR REPLACE FUNCTION public.custom_access_token_hook(event jsonb)
  RETURNS jsonb
  LANGUAGE plpgsql
  STABLE
  AS $$
  DECLARE
    claims jsonb;
    user_role text;
    user_employer_id uuid;
  BEGIN
    -- Look up the profile for the user being issued a token.
    SELECT p.role, p.employer_id
      INTO user_role, user_employer_id
      FROM public.profiles p
     WHERE p.user_id = (event ->> 'user_id')::uuid;

    claims := event -> 'claims';

    IF user_role IS NOT NULL THEN
      claims := jsonb_set(claims, '{role}', to_jsonb(user_role));
    END IF;

    IF user_employer_id IS NOT NULL THEN
      claims := jsonb_set(claims, '{employer_id}', to_jsonb(user_employer_id::text));
    END IF;

    RETURN jsonb_set(event, '{claims}', claims);
  END;
  $$;

  -- Grant the auth admin role permission to call this function.
  GRANT EXECUTE ON FUNCTION public.custom_access_token_hook TO supabase_auth_admin;
  GRANT USAGE ON SCHEMA public TO supabase_auth_admin;
  GRANT SELECT ON public.profiles TO supabase_auth_admin;
  ```

  ```ts
  import 'dotenv/config';
  import postgres from 'postgres';
  import { drizzle } from 'drizzle-orm/postgres-js';
  import { sql } from 'drizzle-orm';
  import * as schema from './schema';
  import { SEED_EMPLOYERS } from './seed-data/employers';
  import { SEED_ROLES } from './seed-data/roles';
  import { SEED_COHORTS } from './seed-data/cohorts';
  import { SEED_PHASES } from './seed-data/phases';
  import { SEED_BARRIERS } from './seed-data/barriers';
  import { SEED_INTERNS } from './seed-data/interns';
  import { SEED_QUESTION_SETS } from './seed-data/question-sets';
  import { SEED_PROGRAM_INFO } from './seed-data/program-info';

  const databaseUrl = process.env.DATABASE_URL;
  if (!databaseUrl) {
    console.error('DATABASE_URL is required');
    process.exit(1);
  }

  async function main() {
    const client = postgres(databaseUrl, { max: 1 });
    const db = drizzle(client, { schema });

    console.log('Resetting public schema (truncating all rows)...');
    await client`TRUNCATE TABLE
      public.assessment_submissions,
      public.questions,
      public.question_sets,
      public.intern_employment_outcomes,
      public.intern_entry_barriers,
      public.intern_entry_assessment,
      public.interns,
      public.cohort_phases,
      public.cohorts,
      public.roles,
      public.employers,
      public.phases,
      public.barriers,
      public.program_info,
      public.profiles
      RESTART IDENTITY CASCADE;`;

    console.log('Seeding program_info...');
    await db.insert(schema.programInfo).values({ id: 1, ...SEED_PROGRAM_INFO });

    console.log('Seeding phases...');
    const phaseRows = await db.insert(schema.phases).values(SEED_PHASES).returning();
    const phaseByLabel = new Map(phaseRows.map((p) => [p.label, p.id]));

    console.log('Seeding barriers...');
    const barrierRows = await db.insert(schema.barriers).values(SEED_BARRIERS).returning();
    const barrierByLabel = new Map(barrierRows.map((b) => [b.label, b.id]));

    console.log('Seeding employers...');
    await db.insert(schema.employers).values(SEED_EMPLOYERS);

    console.log('Seeding roles...');
    await db.insert(schema.roles).values(SEED_ROLES);

    console.log('Seeding cohorts...');
    for (const cohort of SEED_COHORTS) {
      const { phaseLabels, ...rest } = cohort;
      await db.insert(schema.cohorts).values(rest);
      const phaseRows = phaseLabels.map((label, idx) => {
        const phaseId = phaseByLabel.get(label);
        if (!phaseId) throw new Error(`Phase not found: ${label}`);
        return { cohortId: cohort.id, phaseId, sortOrder: idx + 1 };
      });
      if (phaseRows.length > 0) {
        await db.insert(schema.cohortPhases).values(phaseRows);
      }
    }

    console.log('Seeding interns...');
    for (const intern of SEED_INTERNS) {
      const { barrierLabels, entryNotes, ...rest } = intern;
      await db.insert(schema.interns).values(rest);
      await db.insert(schema.internEntryAssessment).values({
        internId: intern.id,
        notes: entryNotes,
        completedAt: new Date(),
      });
      const barrierRows = barrierLabels.map((label) => {
        const barrierId = barrierByLabel.get(label);
        if (!barrierId) throw new Error(`Barrier not found: ${label}`);
        return { internId: intern.id, barrierId };
      });
      if (barrierRows.length > 0) {
        await db.insert(schema.internEntryBarriers).values(barrierRows);
      }
      await db.insert(schema.internEmploymentOutcomes).values({ internId: intern.id });
    }

    console.log('Seeding question sets...');
    for (const qset of SEED_QUESTION_SETS) {
      const { questions, ...rest } = qset;
      await db.insert(schema.questionSets).values({
        ...rest,
        cohortId: rest.cohortId ?? null,
        internId: rest.internId ?? null,
        allowMultiple: rest.allowMultiple ?? false,
        lastEditedAt: new Date(),
      });
      if (questions.length > 0) {
        await db.insert(schema.questions).values(
          questions.map((q) => ({ ...q, questionSetId: qset.id })),
        );
      }
    }

    console.log('Seed complete.');
    await client.end();
  }

  main().catch((err) => {
    console.error(err);
    process.exit(1);
  });
  ```

  ```ts
  import 'dotenv/config';
  import postgres from 'postgres';
  import { drizzle } from 'drizzle-orm/postgres-js';
  import * as schema from './schema';
  import { SEED_PHASES } from './seed-data/phases';
  import { SEED_BARRIERS } from './seed-data/barriers';
  import { SEED_PROGRAM_INFO } from './seed-data/program-info';
  import { SEED_QUESTION_SETS } from './seed-data/question-sets';

  const databaseUrl = process.env.DATABASE_URL;
  if (!databaseUrl) {
    console.error('DATABASE_URL is required');
    process.exit(1);
  }

  async function main() {
    const client = postgres(databaseUrl, { max: 1 });
    const db = drizzle(client, { schema });

    console.log('Idempotent prod seed: only inserts rows that do not yet exist.');

    // program_info: insert if not present
    const existingProgramInfo = await db.select().from(schema.programInfo);
    if (existingProgramInfo.length === 0) {
      await db.insert(schema.programInfo).values({ id: 1, ...SEED_PROGRAM_INFO });
      console.log('Inserted program_info.');
    } else {
      console.log('program_info already exists; skipping.');
    }

    // phases
    const existingPhases = await db.select().from(schema.phases);
    if (existingPhases.length === 0) {
      await db.insert(schema.phases).values(SEED_PHASES);
      console.log(`Inserted ${SEED_PHASES.length} phases.`);
    } else {
      console.log(`${existingPhases.length} phases already exist; skipping.`);
    }

    // barriers
    const existingBarriers = await db.select().from(schema.barriers);
    if (existingBarriers.length === 0) {
      await db.insert(schema.barriers).values(SEED_BARRIERS);
      console.log(`Inserted ${SEED_BARRIERS.length} barriers.`);
    } else {
      console.log(`${existingBarriers.length} barriers already exist; skipping.`);
    }

    // question sets: only standard + competency-core. Cohort/intern overlays are dev-only.
    const standardQSets = SEED_QUESTION_SETS.filter(
      (q) => q.kind === 'standard' || q.kind === 'competency-core',
    );
    for (const qset of standardQSets) {
      const existing = await db.select().from(schema.questionSets).where(
        sql_eq(schema.questionSets.id, qset.id),
      );
      if (existing.length === 0) {
        const { questions, ...rest } = qset;
        await db.insert(schema.questionSets).values({
          ...rest,
          cohortId: null,
          internId: null,
          allowMultiple: rest.allowMultiple ?? false,
          lastEditedAt: new Date(),
        });
        if (questions.length > 0) {
          await db.insert(schema.questions).values(
            questions.map((q) => ({ ...q, questionSetId: qset.id })),
          );
        }
        console.log(`Inserted question set: ${qset.id}`);
      } else {
        console.log(`Question set ${qset.id} exists; skipping.`);
      }
    }

    await client.end();
    console.log('Prod seed complete.');
  }

  // tiny helper to avoid importing eq from drizzle just for one usage:
  function sql_eq<T>(col: T, val: unknown) {
    // Re-import properly:
    // eslint-disable-next-line @typescript-eslint/no-require-imports
    const { eq } = require('drizzle-orm');
    return eq(col, val);
  }

  main().catch((err) => {
    console.error(err);
    process.exit(1);
  });
  ```

  ```ts
  import { createServerClient, parseCookieHeader, serializeCookieHeader } from '@supabase/ssr';
  import type { SupabaseClient } from '@supabase/supabase-js';
  import { env } from './env.server';

  export type AppRole = 'admin' | 'employer';

  export interface AuthContext {
    role: AppRole;
    employerId: string | null;
  }

  /**
   * Pull the role + employer_id JWT claims out of a parsed payload.
   * Returns null for missing/unknown claims.
   */
  export function decodeRoleFromJwtPayload(payload: unknown): AuthContext | null {
    if (!payload || typeof payload !== 'object') return null;
    const obj = payload as Record<string, unknown>;
    const role = obj['role'];
    if (role !== 'admin' && role !== 'employer') return null;
    const employerId = typeof obj['employer_id'] === 'string' ? (obj['employer_id'] as string) : null;
    return { role, employerId };
  }

  /**
   * Create a Supabase server client bound to the current request's cookies.
   * Use this in loaders/actions to read auth state.
   */
  export function createSupabaseServerClient(request: Request, headers: Headers): SupabaseClient {
    const cookies = parseCookieHeader(request.headers.get('Cookie') ?? '');
    return createServerClient(env.SUPABASE_URL, env.SUPABASE_ANON_KEY, {
      cookies: {
        getAll() {
          return cookies;
        },
        setAll(toSet) {
          for (const { name, value, options } of toSet) {
            headers.append('Set-Cookie', serializeCookieHeader(name, value, options));
          }
        },
      },
    });
  }

  /**
   * Get the current user's auth context (role + employer_id) by parsing the JWT.
   * Returns null if no valid session.
   */
  export async function getAuthContext(
    request: Request,
    headers: Headers,
  ): Promise<AuthContext | null> {
    const supabase = createSupabaseServerClient(request, headers);
    const {
      data: { session },
    } = await supabase.auth.getSession();
    if (!session) return null;

    // The access token is a JWT; we decode the payload (no signature verification needed
    // since Supabase already validated the session above).
    const tokenParts = session.access_token.split('.');
    if (tokenParts.length !== 3) return null;
    let payload: unknown;
    try {
      const base64 = tokenParts[1].replace(/-/g, '+').replace(/_/g, '/');
      payload = JSON.parse(Buffer.from(base64, 'base64').toString('utf-8'));
    } catch {
      return null;
    }
    return decodeRoleFromJwtPayload(payload);
  }
  ```

  ```ts
  import 'dotenv/config';
  import postgres from 'postgres';

  /**
   * Open a connection that simulates an authenticated user with the given role + claims.
   * We use Postgres's request.jwt.claims setting to inject claims that auth.jwt() reads.
   */
  export async function withClaims<T>(
    claims: Record<string, unknown>,
    fn: (sql: postgres.Sql) => Promise<T>,
  ): Promise<T> {
    const databaseUrl = process.env.DATABASE_URL;
    if (!databaseUrl) throw new Error('DATABASE_URL required');

    const sql = postgres(databaseUrl, { max: 1 });
    try {
      // The 'authenticated' role exists in Supabase Postgres. Switch to it for RLS to apply.
      await sql`SELECT set_config('request.jwt.claims', ${JSON.stringify(claims)}, false)`;
      await sql`SET LOCAL ROLE authenticated`;
      return await fn(sql);
    } finally {
      await sql.end();
    }
  }

  export const ADMIN_CLAIMS = (userId: string) => ({
    sub: userId,
    role: 'admin',
  });

  export const EMPLOYER_CLAIMS = (userId: string, employerId: string) => ({
    sub: userId,
    role: 'employer',
  ```

  ```ts
  import { db } from './db.server';
  import { interns } from '../../db/schema';
  import { and, eq, isNull, sql } from 'drizzle-orm';

  export interface InternIdentity {
    firstInitial: string;
    lastName: string;
    cohortId: string;
  }

  export interface InternRecord {
    id: string;
    firstInitial: string;
    lastName: string;
    cohortId: string;
  }

  export async function lookupInternByIdentity(
    identity: InternIdentity,
  ): Promise<InternRecord | null> {
    const rows = await db
      .select({
        id: interns.id,
        firstInitial: interns.firstInitial,
        lastName: interns.lastName,
        cohortId: interns.cohortId,
      })
      .from(interns)
      .where(
        and(
          sql`lower(${interns.firstInitial}) = lower(${identity.firstInitial})`,
          sql`lower(${interns.lastName}) = lower(${identity.lastName})`,
          eq(interns.cohortId, identity.cohortId),
          isNull(interns.deletedAt),
        ),
      )
      .limit(1);

    return rows[0] ?? null;
  }
  ```
